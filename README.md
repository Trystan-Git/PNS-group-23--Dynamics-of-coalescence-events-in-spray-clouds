# PNS groep 23: Dynamics of coalescence events in spray clouds
The code for our project can be found in the main branch of our repository, Same as our Manimations code.

All of our data can be found in the folder named 'Data'. In there the data is sorted by the day it was gathered. 
In some folders the data is also sorted by what parameter we were varying.

Our date files after day 3 are named as follows: (height of aperatus)_(nozzle ID)_(flow rate).

The code relevant for our data-analysis can be found in the folder named 'Data analysis'. In there the data analysis is sorted into a folder with some tools we used to be able to analyze the data and by the day where the data was gathered. 

The code relevant for our Manimations can be found in the folder named 'Manim'. 

Our lab journal including log and our meetings including agenda and minutes can be found in our Overleaf project.
